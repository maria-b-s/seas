Use this service to complete your standard or enhanced DBS check application.

What you'll need

Before you start make sure you have:

  • your addresses for the last 5 years and the dates you lived there
  • details of any other names you've used and the dates you used them

If you have them, you will also need your:

  • passport, this can be a UK or non-UK passport
  • National Insurance number
  • UK driving licence

You’ll need a GOV.UK account to use this service. If you do not have a GOV.UK account, you can create one.
